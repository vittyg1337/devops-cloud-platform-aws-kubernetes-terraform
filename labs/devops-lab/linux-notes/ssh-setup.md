# SSH Remote Administration Lab

## Objective
Connect from Windows PowerShell to an Ubuntu Server VM using SSH.

## Tasks Completed

- Installed Ubuntu Server
- Configured networking
- Installed OpenSSH Server
- Verified SSH service status
- Determined VM IP address
- Connected remotely from Windows PowerShell
- Executed Linux commands remotely

## Commands Used

ip addr

sudo systemctl status ssh

ssh username@ip-address
