# Docker WebApp Project

## Objective

Deploy a simple web application using Docker and Nginx on Ubuntu Server.

## Technologies

- Ubuntu Server 24.04
- Docker
- Nginx
- Git
- GitHub
- SSH

## Skills Demonstrated

- Container deployment
- Docker image creation
- Port mapping
- Volume mounting
- Remote administration
- Linux command line
- Git version control

## Result

Successfully deployed a web application accessible over the network through a Docker container.
